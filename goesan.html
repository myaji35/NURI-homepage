핵심 기술: 누리셀 3세대++ (Farm8 대비 경쟁 우위)

1. LangGraph Multi-Agent System (AI Brain)

기존 Farm8: 단순 Rule 기반 자동화 (If-Then 로직)

누리셀 3세대+: 4개 전문 에이전트 협업 시스템 (BMAD Elite 4)

  • Planner (🧠): 생육 계획, 수확 예측, 시장 분석
  • Architect (📐): 환경 제어 로직, 센서 네트워크 최적화
  • Builder (🔨): 자동화 스크립트 구현, 데이터 수집
  • Critic (): 병충해 조기 진단, 이상 탐지, 품질 검증

→ 결과: Rule → Agent (자율 판단 및 학습)

2. GraphRAG Knowledge System (Memory)

기존 Farm8: 단순 시계열 데이터 저장 (MySQL/PostgreSQL)

누리셀 3세대+: Neo4j 기반 지식 그래프 (GraphRAG)

  • 작물 간 연관 관계 학습 (예: 병풀 + 바질 동시 재배 시너지)
  • 병충해 패턴 자동 추론 (Graph Neural Network)
  • 생육 이력 기반 최적 환경 추천

→ 결과: Data → Knowledge Graph (관계 기반 추론)

3. ESP32 + LoRa Mesh Network (Wireless Nerves)

기존 Farm8: 유선 센서 네트워크 (시공비 높음)

누리셀 3세대+: 무선 메시 네트워크 (ESP32 + LoRa)

  • 시공비 70% 절감 (케이블 불필요)
  • 유지보수 간편 (모듈 교체 용이)
  • 확장성 극대화 (새 센서 무선 추가)

→ 결과: 유선 → 무선 메시 (시공비 70% 절감)

4. MLOps + Process Mining (Real-time Optimization)

기존 Farm8: 사후 분석 (주간/월간 리포트)

누리셀 3세대+: 실시간 MLOps 파이프라인 (PM4Py)

  • 모델 자동 재학습 (매일 밤 12시)
  • 작업 프로세스 병목 탐지 (Process Mining)
  • 실시간 환경 제어 피드백 루프

→ 결과: 사후 분석 → 실시간 최적화 (즉시 대응)

LED 조명 일체형

각 원형 포트에 LED 풀스펙트럼 조명이 내장되어 있습니다. 작물별 최적 파장(청색 450nm, 적색 660nm)을 AI가 자동 조절하여 광합성 효율을 극대화합니다.

양액 순환 시스템

각 층마다 독립 양액 공급 라인이 설치되어 있으며, 사용하지 않은 양액은 하단 탱크로 회수되어 재사용됩니다. 물 사용량 80% 절감 효과.

AGV 자동 이송 시스템

수확 시점이 되면 AGV(무인운반차)가 누리셀을 통째로 작업동으로 자동 이송합니다. 발달장애·경계성장애인 근로자가 밝고 쾌적한 작업동에서 편안하게 수확·포장 작업을 수행합니다. 랙 환경의 압박감 제거.

누리셀 시스템 영상 자료

누리셀 적층형 랙 시스템의 핵심 구성 요소와 조립 과정을 영상으로 확인하세요. 실제 작동 원리와 장애인 친화적 설계를 직접 확인할 수 있습니다.

APT-300C 조립 과정

누리셀 랙 프레임의 핵심인 APT-300C 알루미늄 프로파일 조립 과정입니다. 모듈형 설계로 공구 없이 쉽게 조립할 수 있어 유지보수가 간편합니다. 장애인 근로자도 참여 가능한 단순 작업으로 구성되어 있습니다.

Cell-1000 C1 작동 원리

누리셀 1000 C1 모델의 실제 작동 모습입니다. LED 조명, 양액 순환, 온도 제어가 하나의 유닛에 통합되어 있으며, AI가 각 셀을 독립 제어하여 최적의 생육 환경을 유지합니다.

서브 드라이브 시스템

누리셀을 작업동으로 이동시키는 AGV(무인운반차) 서브 드라이브 시스템입니다. 수확 시점이 되면 자동으로 셀을 이동시켜, 장애인 근로자가 밝고 쾌적한 작업동에서 편안하게 작업할 수 있습니다.
